Ashigaru demo

Once the page is loaded, service.js linked from this page will use Ashigaru to send bogus data to the Payment gateway. This results in a new payment method, and the hidden iframe will be redirected to success.html which contains a script tag with our JSON data. The received JSON data will be logged to console.

Open Firebug or any developer tools you may have and monitor the output. If you haven't had ti open, open it now and refresh the page.

Get Ashigaru

Ashigaru has not yet been released. You can currently obtain the development version from the Daimyo Github repository in either uncompressed or compressed form.
